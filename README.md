# EcoVista

EcoVista is a Next.js application designed to provide weather-related information, including air quality, temperature, wind, and general weather conditions. The project is built using modern web technologies such as React, TypeScript, and TailwindCSS.

## Features

- **Weather Information**: Displays weather details such as temperature, wind speed, and air quality.
- **Location Detection**: Automatically detects the user's location to provide localized weather data.
- **Modular Components**: Reusable and maintainable React components.
- **TypeScript Support**: Ensures type safety and better developer experience.
- **TailwindCSS Integration**: For rapid and responsive UI development.

## Folder Structure

The project follows a structured and modular architecture:

```
.ecovista/
├── .env                     # Environment variables
├── .gitignore               # Git ignore file
├── eslint.config.mjs        # ESLint configuration
├── next.config.ts           # Next.js configuration
├── package.json             # Project dependencies and scripts
├── postcss.config.mjs       # PostCSS configuration for TailwindCSS
├── tsconfig.json            # TypeScript configuration
├── app/                     # Next.js app directory
│   ├── globals.css          # Global styles
│   ├── layout.tsx           # Root layout
│   ├── page.tsx             # Home page
│   └── [location]/          # Dynamic routes for location-specific data
│       ├── layout.tsx       # Layout for location pages
│       └── page.tsx         # Page for location-specific weather data
├── components/              # Reusable React components
│   ├── AQIComponent.tsx     # Air Quality Index component
│   ├── Card.tsx             # Generic card component
│   ├── LocationDetector.tsx # Location detection component
│   ├── LocationInfo.tsx     # Location information display
│   ├── TemperatureComponent.tsx # Temperature display component
│   ├── WeatherComponent.tsx # Weather details component
│   └── WindComponent.tsx    # Wind details component
├── lib/                     # Utility functions and libraries
│   ├── location-info.ts     # Location-related utilities
│   └── weather-info.ts      # Weather-related utilities
├── public/                  # Static assets (images, icons, etc.)
└── .next/                   # Build output (generated by Next.js)
```

## Getting Started

### Prerequisites

Ensure you have the following installed:

- **Node.js** (v16 or higher)
- **npm** (v8 or higher)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/ecovista.git
   cd ecovista
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Set up environment variables:

   Create a `.env` file in the root directory and add the required environment variables (e.g., API keys for weather data).

4. Run the development server:

   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

The following scripts are available in the `package.json`:

- **`npm run dev`**: Starts the development server.
- **`npm run build`**: Builds the application for production.
- **`npm run start`**: Starts the production server.
- **`npm run lint`**: Runs ESLint to check for code quality issues.

## Environment Variables

The project uses a `.env` file to manage environment-specific configurations. Add the following variables:

```env
NEXT_PUBLIC_WEATHER_API_KEY=your_api_key
NEXT_PUBLIC_MAP_API_KEY=your_map_api_key
```
